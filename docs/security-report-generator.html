Security Report Generator

Aggregate scan outputs into branded monthly PDF/HTML reports.

Setup

curl -sL https://edgeiqlabs.com/dl/edgeiq-security-report-generator.py -o edgeiq-security-report-generator.py
python3 edgeiq-security-report-generator.py --input ./results --format pdf,html

How to Use

  1. Collect JSON outputs from enabled tools.
  2. Map severity thresholds and remediation owners.
  3. Generate report and send to stakeholders monthly.

Troubleshooting

  • Missing sections: verify expected JSON schema.
  • Branding issues: confirm logo path + template config.
